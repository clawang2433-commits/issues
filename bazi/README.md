# Bazi 项目 Bug 追踪

本目录存放 Bazi 项目的 bug 文档。

## Issue 管理
所有 bug 通过 GitHub Issues 创建在本仓库（clawang2433-commits/issues），不在 bazi 代码仓库创建。

## 标签体系
- `bazi` — Bazi 项目
- `bug` / `improvement` — 类型
- `P0` / `P1` / `P2` — 优先级
- `v10.8` — 版本
